👑 MCEnginePremiumApi – Rank Management

🆕 createPremiumRank(String rankType)

Creates a new premium rank table if it doesn't exist.

Usage:

MCEnginePremiumApi api = MCEnginePremiumApi.getApi();
api.createPremiumRank("vip");
api.createPremiumRank("vvip");
api.createPremiumRank("mvp");

🔍 getPremiumRank(String uuid, String rankType)

Returns the current rank level of the player (or -1 if not found).

Usage:

MCEnginePremiumApi api = MCEnginePremiumApi.getApi();
String uuid = "player-uuid";
int vipRank = api.getPremiumRank(uuid, "vip");
int vvipRank = api.getPremiumRank(uuid, "vvip");

⬆️ upgradePremiumRank(String uuid, String rankType)

Increases the player’s rank level by 1 for the given rank type.

Usage:

MCEnginePremiumApi api = MCEnginePremiumApi.getApi();
String uuid = "player-uuid";
api.upgradePremiumRank(uuid, "vip");
api.upgradePremiumRank(uuid, "vvip");
