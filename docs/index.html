📘 MCEnginePremiumApi Usage

🔧 Initialization

Creates the API instance. Automatically chooses database (MySQL or SQLite) from config.yml.

Plugin plugin = ...;
MCEnginePremiumApi api = new MCEnginePremiumApi(plugin);

🔗 Get API Instance

Retrieves the singleton instance of the API.

MCEnginePremiumApi api = MCEnginePremiumApi.getApi();

📦 Get Plugin Reference

Returns the associated plugin object.

Plugin plugin = api.getPlugin();

💾 Get Database Implementation

Access the active database connection (MySQL/SQLite) implementing IMCEnginePremiumApiDB.

IMCEnginePremiumApiDB db = api.getDB();

🆕 Create Premium Rank Table

Initializes the table if it doesn't exist. Supports types like "vip", "vvip", "mvp", etc.

api.createPremiumRank("vip"); api.createPremiumRank("vvip");

🔍 Get Player's Premium Rank

Returns the numeric rank of a player for a specific rank type.

String uuid = "player-uuid";
int vipRank = api.getPremiumRank(uuid, "vip");
int vvipRank = api.getPremiumRank(uuid, "vvip");

⬆️ Upgrade Player's Rank

Increases the player's rank by 1 in the given rank type.

api.upgradePremiumRank("player-uuid", "vip");
api.upgradePremiumRank("player-uuid", "vvip");
